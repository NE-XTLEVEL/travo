## 0. Introduction

Travo는 여행 계획에 어려움을 느끼는 사용자들을 위한 프롬프트 기반 LLM 여행 계획 추천 서비스입니다. 사용자는 자연어로 여행에 대한 간단한 요구사항만 입력하면, Travo가 맞춤형 일정을 자동으로 구성하여 추천해줍니다.
